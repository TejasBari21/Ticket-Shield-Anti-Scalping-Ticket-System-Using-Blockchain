# 📌 TicketShield – Anti-Scalping Ticket System Using Blockchain

## Project Overview

TicketShield is a decentralized ticketing system built using Blockchain technology to prevent ticket scalping, fraud, and unauthorized reselling. It ensures transparent, secure, and fair ticket distribution using smart contracts.

The system uses blockchain-based ownership records so every ticket is verifiable, tamper-proof, and cannot be illegally duplicated or resold.

---

## Features

- Anti-scalping protection using blockchain  
- Blockchain-based ticket ownership  
- Fast and transparent verification system  
- Secure transactions through smart contracts  
- Immutable ticket records  
- Fraud-proof ticket generation  
- Transparent ownership tracking  

---

## Problem Statement

Traditional ticketing systems suffer from:

- Fake ticket creation  
- Ticket scalping (reselling at high prices)  
- Lack of transparency  
- Centralized control and fraud risks  

---

## Solution

TicketShield solves these issues by:

- Using blockchain smart contracts for ticket creation and ownership  
- Ensuring immutable ticket records  
- Preventing unauthorized resale  
- Providing a transparent verification system

## How It Works

- User logs into the platform  
- Ticket is created as a blockchain asset  
- Smart contract assigns ownership  
- Ticket is stored on blockchain  
- Verification happens via smart contract  
- Resale is restricted or controlled

## Installation & Setup

Clone Repository

git clone https://github.com/TejasBari21/TejasBari21-Ticket-Shield-Anti-Scalping-Ticket-System-Using-Blockchain.git

Install Dependencies

cd TicketShield  
npm install  

Run Backend

cd backend  
npm start  

Run Frontend

cd frontend  
npm start
